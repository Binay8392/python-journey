## 📖 Python Theory: The Basics

### What is Python?
Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in web development, data science, automation, artificial intelligence, and more.

### Key Concepts
- **Variables**: Containers for storing data values (e.g., numbers, text).
- **Data Types**: Common types include integers (`int`), floating-point numbers (`float`), strings (`str`), and booleans (`bool`).
- **Operators**: Symbols that perform operations on variables and values (e.g., `+`, `-`, `*`, `/`).
- **Conditionals**: Allow your program to make decisions using `if`, `elif`, and `else`.
- **Loops**: Repeat actions using `for` and `while` loops.
- **Functions**: Blocks of code that perform a specific task and can be reused.
- **Input/Output**: Interact with users using `input()` and display results with `print()`.

### Example: Hello World
```python
print("Hello, World!")
```
